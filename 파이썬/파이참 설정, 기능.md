### 파이참 가상 환경 설정
---

#### 가상환경
- 프로젝트마다 필요한 모듈, 혹은 버전을 다르게 하기 위해 사용

#### 설정법
- file - project - add-interpreter를 통해 가상 환경 생성
- 아래에 터미널 선택 -> 터미널 목록에서 + 옆에 있는 아래 화살표 클릭
- Command Prompt 선택
- 이곳에서 pip 명령어들 사용 가능

---
### 파이참 유용한 기능
---

#### 검색 기능
- F4: 선택한 변수(함수)를 누르고 F4를 누를 시 선언부로 이동(파일간에 이동도 됨)
- Shift + ctrl + f: 코드를 전역으로 검색
